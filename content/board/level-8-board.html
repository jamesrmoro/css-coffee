The coffee cups need to stay in the same order as the saucers using flex-direction. This CSS property defines the direction in which the items are positioned in the container and accepts the following values:

  • row: Items are positioned in the same direction as the text.
  • row-reverse: Items are positioned in the opposite direction of the text.
  • column: Items are positioned from top to bottom.
  • column-reverse: Items are positioned from bottom to top.
.table {
	display: flex;
}
